# Single-Cycle_CPU
-Implementation of a Single-Cycle CPU in VHDL.<br />
-Instruction per Cycle.<br />
-Control unit based on a finite state machine.<br />

## Built With
-Quartus Prime Version 20.1.1 Build 720 11/11/2020 SJ Lite Edition<br />
